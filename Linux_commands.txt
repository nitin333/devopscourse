1- ls (list of contents in a folder)
2- cd <folder_name> (change directory from present)
3- cd <folder_name>/ (go inside the sample folder)
4- touch <filename> (creates a file)
5- cd .. (move out of current directory)
6- mkdir <demo> (creates a new directory)
7- cp <filename> ./<foldername>/ (copy a file inside folder)
8- mv <filename> ./<foldername>/ (move a file from folder)
9- rm <filename> (delets a file)
10- rm -r ./<foldername>/ (remove a directory)
